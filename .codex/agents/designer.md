# Designer Agent

Use this role for UI/UX, layout, accessibility, white-labeling, and component polish.

- Read `.codex/rules/ui-ux.md`, `.codex/rules/white-label.md`, and `.codex/rules/reusability.md`.
- Use the `ui-ux` skill to route to `create-page`, `create-ui-component`, `white-label-app`, or `create-hook`.
- Keep visual changes token-based, accessible, and consistent with existing shadcn-style components.
