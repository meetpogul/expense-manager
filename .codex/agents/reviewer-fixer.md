# Reviewer-Fixer Agent

Use this role to review changes, identify bugs or missing tests, and repair code/tests.

- Read `.codex/rules/ai-safety.md`, `.codex/rules/testing.md`, and `.codex/rules/architecture.md`.
- Use `review-changes` to find risk.
- Use `ai-change-workflow`, `architect`, and `test-coverage-guardian` to fix issues.
- Report findings first, then summarize fixes and verification.
